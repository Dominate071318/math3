Chapter 1 Notes

Section 1-1

Algebraic Expressions

Learn the vocabulary — these words translate English into math.

Additionsum, plus, more than
Subtractiondifference, minus, less than
Multiplicationproduct, times
Divisionquotient, divided by
  • Algebraic expression: has variables (x, y, a, b) AND numbers AND operations
  • Numerical expression: numbers and operations only — NO variables
  • "32 more than n" → n + 32
  • "58 less than n" → n − 58 (NOT 58 − n!)
  • "6 times a number n" → 6n
  • "Quotient of n and 5" → n/5
  • "3 more than twice x" → 2x + 3
Section 1-2

Evaluating Expressions & PEMDAS

Evaluating = simplifying (no = sign, not solving). Combine like terms. Follow PEMDAS!

PEMDAS: Parentheses → Exponents → Mult/Div → Add/Sub
  • Base = the big number; Exponent = how many times you multiply it by itself
  • 2⁵ = 2·2·2·2·2 = 32
  • 3⁴ = 3·3·3·3 = 81
  • Fractions raised to a power get SMALLER: (½)³ = 1/8
  • If x=5, y=2: evaluate x²+x−12÷y² → 25+5−3 = 27
  • Always show your work step by step!
Section 1-3

Square Roots

A square root asks: what number times itself equals this? The number inside the √ is called the radicand.

√49 = 7 because 7 × 7 = 49 (Perfect Square!)
  • Perfect squares give whole number answers: √64=8, √100=10, √36=6
  • Non-perfect squares give decimals: √48≈6.93
  • To estimate: find nearest perfect squares above and below
  • √178 ≈ 13.34 → rounds to 13
  • Rounding: 0.5 and above → round up; below 0.5 → round down
  • √(25/49) = √25/√49 = 5/7
  • To order: convert everything to decimals first!
Section 1-4

Properties of Numbers

These properties explain WHY math works — you use them without thinking!

  • Commutative (Add): a+b = b+a → order doesn't matter
  • Commutative (Mult): a·b = b·a → order doesn't matter
  • Associative (Add): (a+b)+c = a+(b+c) → grouping doesn't matter
  • Associative (Mult): (a·b)·c = a·(b·c) → grouping doesn't matter
  • Mult. Property of −1: −1·a = −a → multiplying by negative flips the sign
  • + × + = +  |  + × − = −  |  − × − = +
Section 1-5

The Number Line & Absolute Value

Positive numbers are always to the RIGHT. Negative numbers are always to the LEFT.

Absolute Value = how far from 0. ALWAYS POSITIVE!
  • |4| = 4    |−4| = 4
  • Additive Inverse: what gets you back to 0 (−10 + 10 = 0)
  • Adding fractions: must have a COMMON denominator first
  • Whatever you do to the bottom of a fraction, do to the top too!
  • 3 + (−5) = −2 → same as 3 − 5
Section 1-6

Multiplying & Dividing Real Numbers

Sign rules are everything here. Memorize the sign chart!

  • + × + = +  |  + × − = −  |  − × + = −  |  − × − = +
  • Multiplying fractions: multiply straight across (top × top, bottom × bottom)
  • Dividing fractions: flip the second fraction (reciprocal), then multiply
  • Multiplicative Inverse: what fraction × original = 1? (e.g. 10 × 1/10 = 1)
  • Simplify and reduce after multiplying!
Section 1-7

Distributive Property

Multiply the OUTER term by everything INSIDE the parentheses.

a(b + c) = ab + ac
  • 3(x + 8) = 3x + 24
  • −7(5x − 4) = −35x + 28
  • −(2y − 3x) = −2y + 3x (multiply by −1, flips all signs!)
  • 1x = x (the 1 is always there, doesn't need to be written)
  • After distributing, combine like terms
  • Simplify fractions: 4−2x/8 → ½ − x/4
Section 1-8

Equations: True, False, or Open?

Check both sides — do they equal? A variable with no given value = Open.

  • True: both sides equal → 24+18 = 20+22 → 42=42 ✓ TRUE
  • False: sides don't equal → 14−7 = 11+6 → 7≠17 FALSE
  • Open: has a variable with no given value → 2x−10=24
  • If a value IS given for x, plug it in and check — it becomes True or False!
  • 6x−8=−5; x=½ → 6(½)−8=−5 → 3−8=−5 → −5=−5 TRUE
Section 1-9

Ordered Pairs & Solutions

An ordered pair (x, y) — first number is ALWAYS x, second is ALWAYS y. To check if it's a solution, substitute both values and see if the equation is true.

  • Is (5, 20) a solution to y = 4x? → 20 = 4(5) → 20=20 YES!
  • Is (6, 18) a solution to y−6=x+2? → 18−6=6+2 → 12≠8 NO
  • Always show your work — substitute both values!
  • Is (−10,−2) a solution to x/5=y? → −10/5=−2 → −2=−2 YES
  • Is (5,1) a solution to y=11−2x? → 1=11−10 → 1=1 YES
  • ★ Always show your work ★
