Chapter 4 Notes

Section 4-1

Graphs Relating Quantities

Graphs show the relationship between two things. Always LABEL your graph — the labels are the variables (x and y)!

  • Balloon: goes up when blowing, drops when taking breaths, flat when stopped, pops!
  • Cutting a board: length goes DOWN a little each time you cut, stays flat while measuring
  • Cell phone: flat rate for first 150 mins, then sharp increase (per minute charges)
  • Rocket: rises fast, slows when fuel runs out, falls when parachute opens, hits ground
  • Always ask: what's on the x-axis (time usually) and y-axis (the thing changing)?
Section 4-2

Linear Tables, Equations & Graphs

When x and y both change at a CONSTANT rate, it's LINEAR. You can make a table, graph it, and write an equation.

y = (constant rate) × x + (y-value when x=0)
  • Look at the table — does y change by the same amount each time? → LINEAR
  • * Constant rate of change = multiply x by that number
  • ** y-intercept = whatever y equals when x = 0
  • Example: perimeter of rectangles → y = 2x + 12
  • If rate is NOT constant → NON-LINEAR (just write that word!)
  • Check your equation by plugging in an x value from the table
Section 4-3

Linear vs. Non-Linear Functions

Constant rate of change = LINEAR. Not constant = NON-LINEAR. It's that simple!

  • LINEAR graphs: straight lines (going up, down, horizontal, or diagonal)
  • NON-LINEAR graphs: curves — parabola (x²), absolute value (V-shape), x³ curve
  • To check a table: calculate the change between each y value
  • If changes are all the same → LINEAR  |  If different → NON-LINEAR
  • x-axis: left = negative, right = positive
  • y-axis: down = negative, up = positive
Section 4-4

Graphing Equations (x/y Chart)

To graph any equation, make an x/y chart. Use x = −2, −1, 0, 1, 2 every time. Solve for y with each x value.

  • ① Make an x/y chart
  • ② Use x = −2, −1, 0, 1, 2 as your x values
  • ③ Plug each x into the equation and solve for y
  • ④ Plot each (x, y) point on the graph
  • ⑤ Connect the dots with a straight line
  • Example: y = −2x + 1 → x=−2 gives y=5, x=0 gives y=1, x=2 gives y=−3
  • Absolute value equations create V-shaped graphs!
Section 4-6

Domain, Range & Functions

Domain = all the X values. Range = all the Y values. Think of it as (D, R) just like (x, y).

Domain = X values  |  Range = Y values
  • List domains in numerical order (least to greatest)
  • List ranges in numerical order
  • NO repeated numbers — if a number appears twice, only write it ONCE
  • Use a mapping diagram: draw arrows from D to R

Is it a FUNCTION?

✅ 2 domains → 1 range: YES
❌ 1 domain → 2 ranges: NO
✅ 1 domain → 1 range: YES
  • Vertical Line Test: if ANY vertical line crosses the graph more than once → NOT a function
  • Fails the test ONCE → not a function at all!
