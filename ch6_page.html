Chapter 6 Notes

Section 6-1

Graphing Systems of Equations

A system of equations is TWO equations. The solution is the point where both lines CROSS — the (x, y) coordinate they share.

3 Methods: Graphing · Substitution · Elimination
  • Put both equations in slope-intercept form (y = mx + b)
  • Graph both lines — where they cross is the SOLUTION
  • Write the solution as an ordered pair (x, y)
  • Same slope = PARALLEL = NO SOLUTION (lines never meet)
  • Check: plug your answer into BOTH equations — must work in both!
  • Example: y=x+2 and y=3x−2 → cross at (2, 4) → Solution: (2, 4)
Section 6-2

Substitution Method

One equation is already solved for x or y? Plug that expression into the other equation! This is the fastest way when one variable is already isolated.

1Get one equation equal to x or y
2Substitute (replace) into the other equation
3Solve for the variable you get
4Plug that value back in to find the other variable
5Write solution as (x, y)
  • y=3x and x+y=−32 → x+3x=−32 → 4x=−32 → x=−8 → y=−24 → (−8,−24)
  • y=2x+7 and y=x−1 → 2x+7=x−1 → x=−8 → y=−9 → (−8,−9)
Section 6-3 Part 1

Elimination Method

Line the equations up in columns. Add them together to ELIMINATE one variable. Then solve for what's left!

1Line equations up in columns
2Add/subtract to eliminate one variable
3Solve for the remaining variable
4Plug back in to find the other variable
5Write solution as (x, y)
  • 5x−6y=−32 AND 3x+6y=48 → add them → 8x=16 → x=2 → plug in → y=7 → (2,7)
  • Can't eliminate? Multiply one equation by −1 to flip the signs
  • 3x+2y=−15 AND x−2y=19 → add → 4x=4 → x=1 → y=−9 → (1,−9)
Section 6-3 Part 2

More Elimination (Multiply First)

When you can't eliminate right away — multiply one or BOTH equations by a number to create matching coefficients, then eliminate!

  • Ask: what can I multiply by to get one value to eliminate?
  • −2x+15y=−32 AND 7x−6y=17 → multiply 2nd by 3 → 21x−18y=51 → then multiply 1st by 7 → add!
  • May need to multiply BOTH equations
  • 5x+3y=26 AND −2x+7y=6 → multiply: 5x×2=10x, 2x×5=10x → eliminate!
  • Solution: (4, 2)
Section 6-5

Linear Inequalities (Graphing)

Graph like a regular equation, but shade one side. Ask yourself two questions on EVERY problem!

⭐ Ask every time: ① Solid or dashed line? ② Which side is shaded?
  • < or > → DASHED line (not included)
  • ≤ or ≥ → SOLID line (included)
  • To find shading: plug (0,0) into the inequality
  • If TRUE → shade the side where (0,0) is
  • If FALSE → shade the OTHER side (without (0,0))
  • y > x−2: dashed, test (0,0): 0>−2 TRUE → shade where (0,0) is (above line)
Section 6-6

Systems of Linear Inequalities

Graph TWO inequalities on the same graph. Shade each one, then the OVERLAP (where both shadings agree) is the solution!

  • Graph each inequality with its own line (solid or dashed)
  • Shade each one using the (0,0) test
  • The solution is where BOTH shadings overlap
  • Only shade where both lines agree!
  • Example: y < 2x−3 AND y > −2x+2 → graph both, shade overlap region
  • If equations aren't in y=mx+b form, rearrange first!
