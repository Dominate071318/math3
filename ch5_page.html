Chapter 5 Notes

Section 5-1

Slope

Slope = how steep a line is. It's the rate of change — how much y goes up or down for every step right.

Slope = Rise/Run = (y₂ − y₁) / (x₂ − x₁)
  • Rise = how much you go UP or DOWN (vertical)
  • Run = how much you go RIGHT (horizontal)
  • Only 1 negative sign in your fraction = negative slope
  • Horizontal line: slope = 0 (0/anything = 0)
  • Vertical line: slope = undefined (can't divide by 0)
  • Example: points (1,260) and (2,520) → slope = (520−260)/(2−1) = 260
  • Example: (−1,0) and (3,−2) → slope = (−2−0)/(3−(−1)) = −2/4 = −1/2
Section 5-3

Slope-Intercept Form

The most common way to write a line. m = slope, b = where the line crosses the y-axis (when x = 0).

y = mx + b
  • m = slope  |  b = y-intercept (x=0)
  • y = 5x − 2: slope m=5, y-intercept b=−2
  • y = −½x + ⅔: slope m=−½, y-intercept b=⅔
  • Write in slope-intercept: m=−4/5, b=7 → y = −4/5x + 7
  • Graph: start at b on y-axis, then use slope (rise/run) to find next points
  • To graph from equation: identify m and b, plot b, move rise/run
Section 5-4

Point-Slope Form

Use when you know a POINT on the line and the SLOPE. The y and x coordinates in the formula are OPPOSITES (flip their signs)!

y − y₁ = m(x − x₁)
  • Point (3,6), m=−5 → y−6 = −5(x+3) [x is opposite of 3]
  • Point (8,−4), m=⅔ → y+4 = ⅔(x−8)
  • Point (2,−1), m=4 → y+1 = 4(x−2)
  • Read the point from point-slope: y−1 = ⅔(x−2) → point (2,1), slope ⅔
  • 3 steps to write slope-intercept from two points: ① find slope ② pick a point ③ solve for b
Section 5-5

Standard Form

Another way to write a line. All real numbers (no fractions). x stays with x, y stays with y.

Ax + By = C
  • To find x-intercept: replace y with 0, solve for x → (x, 0)
  • To find y-intercept: replace x with 0, solve for y → (0, y)
  • Example: 3x + 4y = 24
  • → y-int: 3(0)+4y=24 → y=6 → (0, 6)
  • → x-int: 3x+4(0)=24 → x=8 → (8, 0)
  • Point-slope → slope-intercept → move x to left side → standard form
Section 5-6

Parallel & Perpendicular Lines

Parallel → SAME slope, different y-intercept
  • Like train tracks — run side by side, never cross
  • y=2x+4 and y=2x−8: both m=2 → PARALLEL
  • To write parallel line: "steal" the slope from the given equation
  • Given (−3,−1) parallel to y=2x+3: steal m=2, use point-slope → y=2x+5
Perpendicular → OPPOSITE RECIPROCAL slopes
  • Cross at 90° angles
  • Flip the fraction AND change the sign
  • m=⅔ → perpendicular slope = −3/2
  • To write perp. line: steal opposite reciprocal slope, use point-slope
  • (2,4) perp to y=⅓x−1: new m=−3, → y=−3x+10
