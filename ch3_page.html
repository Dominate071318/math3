Chapter 3 Notes

Section 3-1 Part A

Writing & Solving Inequalities

An inequality is like an equation but instead of = we use <, >, ≤, or ≥. Think of it like an alligator — the mouth wants to eat the BIGGER number!

<less than
>greater than
≤less than or equal to
≥greater than or equal to
  • "All real numbers x less than or equal to −7" → x ≤ −7
  • "6 less than a number k is greater than 13" → k − 6 > 13
  • Solve like an equation — move everything away from the variable
  • 2x + 1 > −3 → 2x > −4 → x > −2
  • Always read the VARIABLE FIRST when graphing
Section 3-1 Part B

Graphing Inequalities on a Number Line

Draw the solution on a number line. The type of dot and arrow direction matter!

Greater than (>) → arrow points RIGHT  |  Less than (<) → arrow points LEFT
  • OPEN dot (○) for < and > — the number is NOT included
  • CLOSED dot (●) for ≤ and ≥ — the number IS included
  • x > 7: open dot at 7, arrow goes right
  • n ≥ −2: closed dot at −2, arrow goes right
  • x < −15: open dot at −15, arrow goes left
  • "Trail rides begin at $19.99" → x ≥ $19.99
Section 3-2

Adding & Subtracting Inequalities

Same process as solving equations — add or subtract from both sides. The inequality sign doesn't change!

⭐ Always read the VARIABLE FIRST when writing/graphing your answer!
  • x − 15 > −12 → add 15 to both sides → x > 3
  • x − 15 ≥ −12 → same steps → x ≥ 3 (same process!)
  • 10 ≥ x − 3 → add 3 → 13 ≥ x → READ: x ≤ 13 (variable first!)
  • y − 11 ≥ −2 → add 11 → y ≥ 9
Section 3-3

Multiplying & Dividing Inequalities

Same as equations, BUT there's one BIG rule to remember:

⚠️ When you multiply or divide by a NEGATIVE number → FLIP the inequality sign!
  • x/3 < −2 → multiply by 3 → x < −6 (positive, no flip)
  • x/5 ≤ 4 → multiply by 5 → x ≤ 20 (positive, no flip)
  • −4y ≤ 63 → divide by −4 → y ≥ −7 (FLIP! negative divisor)
  • −6x ≥ 30 → divide by −6 → x ≤ −5 (FLIP! negative divisor)
  • Only flip when you × or ÷ by a negative — doesn't matter if answer is negative
Section 3-4

Multi-Step Inequalities

Combine everything you know. Follow these steps every time.

  • Steps: ① Distribute   ② Combine like terms   ③ Move constants   ④ Divide/multiply   ⑤ Graph — READ VARIABLE FIRST!
  • 9 + 4t > 21 → 4t > 12 → t > 3 (open dot, right)
  • 38 ≤ 2(4y+7) → distribute → 38 ≤ 8y+14 → 24 ≤ 8y → 3 ≤ y → y ≥ 3
  • −6x − 7 ≤ 17 → −6x ≤ 24 → x ≥ −4 (FLIP! divided by −6)
  • 6x − 1 > 3x + 8 → 3x > 9 → x > 3
Section 3-6

Compound Inequalities

Two inequalities combined into one. AND = both must be true (between two numbers). OR = either one can be true (goes in two directions).

  • AND problems: −3 ≤ m − 4 < −1 → solve both sides → 1 ≤ m < 3
  • OR problems: solve each inequality separately, then combine
  • Graph AND: closed/open segment between two numbers
  • Graph OR: two arrows going in opposite directions
  • −7 < 3y − 4 < 14 → add 4 to all parts → −3 < 3y < 18 → −1 < y < 6
Section 3-7

Absolute Value Equations & Inequalities

Absolute value = distance from zero. ALWAYS positive! When you solve, you get TWO answers — a positive AND a negative version.

|x| = 4 → x = 4 OR x = −4
  • |3x − 1| = 8 → split: 3x−1=8 AND 3x−1=−8
  • → x = 3 AND x = −7/3
  • Get absolute value ALONE first, then split
  • |x| + 2 = 9 → |x| = 7 → x = 7 or x = −7
  • Absolute value CANNOT equal a negative → NO SOLUTION
|AV| < # → AND problem  |  |AV| > # → OR problem
  • |n − 1| < 2 → AND: n−1<2 AND n−1>−2 → −1<n<3
  • |8x| ≥ 24 → OR: 8x≥24 OR 8x≤−24 → x≥3 OR x≤−3
  • |2x+4| ≥ 5 → OR problem → x ≥ ½ OR x ≤ −9/2
  • To split OR: change the number, flip the sign on the second one
