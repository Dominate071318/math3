Chapter 2 Notes

Section 2-1

One-Step Equations

Get the variable by itself. Whatever you do to one side of the = sign, you MUST do to the other side.

1Leave the variable in place
2Move everything else away from it (do the opposite operation)
3Do the same to BOTH sides
  • x + 4 = −6 → subtract 4 from both sides → x = −10
  • x − 11 = 19 → add 11 to both sides → x = 30
  • 4x = 20 → divide both sides by 4 → x = 5
  • x/3 = 7 → multiply both sides by 3 → x = 21
  • Combine like terms FIRST if needed
Section 2-2

Two-Step Equations

Get the variable by itself. Move the number attached to the variable LAST.

1Move the constant (number alone) first — do the opposite
2Move what is attached to the variable last (divide or multiply)
3Work DOWN rather than across
  • 2x + 3 = 15 → subtract 3 → 2x=12 → divide by 2 → x=6
  • 5x + 1 = 16 → subtract 1 → 5x=15 → divide by 5 → x=5
  • x/8 − 4 = 2 → add 4 → x/8=6 → multiply by 8 → x=48
  • (x−7)/3 = −12 → multiply by 3 → x−7=−36 → add 7 → x=−29
Section 2-3

Multi-Step Equations

Combine like terms first, then solve like a two-step equation.

1Combine like terms (CLT)
2Get variable by itself
3Move what is attached to the variable last
  • −2x + 5 + 5x = 14 → 3x + 5 = 14 → 3x = 9 → x = 3
  • 11y − 8 − 6y = 22 → 5y − 8 = 22 → 5y = 30 → y = 6
  • −8(2x−1) = 36 → distribute first → −16x + 8 = 36 → x = −1.75
  • 4(x−2) + 6 = 0 → distribute → 4x − 2 = 0 → x = 0.5
Section 2-4

Variables on Both Sides

Move all variables to one side, all numbers to the other. Either side works — pick the easier one!

  • 5x + 2 = 2x + 14 → subtract 2x → 3x + 2 = 14 → 3x = 12 → x = 4
  • 7y + 2 = 4y − 10 → subtract 4y → 3y + 2 = −10 → 3y = −12 → y = −4
  • 2(5x−1) = 3(x+11) → distribute both sides first!
  • → 10x − 2 = 3x + 33 → 7x = 35 → x = 5
  • On same side of = sign: just combine, don't change the sign
Section 2-5

Solving for a Variable (Given Values)

When given a value for one variable, plug it in and solve for the other.

  • 10x + 5y = 80, find y when x = 3:
  • → 10(3) + 5y = 80 → 30 + 5y = 80 → 5y = 50 → y = 10
  • When x = 6: 10(6) + 5y = 80 → 60 + 5y = 80 → y = 4
  • 4x = 3y − 5, find y when x = 4,5,6 → plug in each x value separately
Section 2-6

Unit Rate & Ratios

Unit rate = how much for ONE unit. Use division.

Unit Rate = Total ÷ Number of Units
  • $6 for 5 apples → $6 ÷ 5 = $1.20 per apple
  • Gas mileage: miles per gallon
  • Wages: dollars per hour
  • Ratios use division — set up as a fraction
Section 2-7

Solving Proportions

Two equal ratios. Cross multiply, then divide.

a/b = c/d → a·d = b·c
1Cross multiply (diagonal × diagonal)
2Divide to solve for x
  • 7/8 = x/12 → 8x = 84 → x = 10.5
  • x/7 = 4/5 → 5x = 28 → x = 5.6
  • 4/3 = 14/y → 4y = 42 → y = 10.5
  • With algebra in numerator: cross multiply first, then distribute
Section 2-8

Proportions & Similar Figures

~ means SIMILAR (same shape, not same size). Similar figures have proportional sides and equal angles.

  • △ABC ~ △DEF means ∠B~∠E, ∠A~∠D, ∠C~∠F
  • Corresponding sides are proportional: AB/BC = DE/EF
  • Set up a proportion, cross multiply, divide
  • Make sure similar sides are in the SAME spot of each fraction!
  • Example: 16/y = 18/12 → 18y = 192 → y = 24  |  18/x = 24/10 → x = 7.5
